# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-03-15

### Added

- Initial release
- MiniDLNA (ReadyMedia) DLNA/UPnP media server
- Ingress support for web UI
- Direct port access on 8200
- Configurable media directories with type prefixes (A/V/P/AV)
- inotify support for automatic media discovery
- Force rescan option for rebuilding media database
- Subtitle support (enable_subtitles)
- Configurable log levels
- Wide links support for symlinks outside media directories
- Root container selection (browse directory/music/video/pictures)
- Health check for container monitoring
- Multi-architecture support (amd64, i386, aarch64, armv7, armhf)
